EcoWear AI

An intelligent and eco-conscious web application that helps users track the environmental impact of fashion products by analyzing **eco score** and **carbon footprint**.

Project Description

EcoWear AI is a smart platform designed to promote **sustainable fashion choices**.  
Users can add clothing items, analyze their environmental impact, and make better decisions using real-time insights.

The system calculates:
- 🌱 Eco Score (based on material)
- 🌍 Carbon Footprint (based on material + price)

It also allows users to combine items and analyze the impact of an entire outfit.
## ✨ Features

### 🔹 Core Features
- Add products/outfits with details
- Upload product images
- Real-time eco & carbon calculation
- Live preview before adding product

### 🔹 Smart Insights
- Eco Score calculation (Cotton, Polyester, Recycled)
- Carbon footprint estimation
- Recommendation system:
  - Eco-friendly 🌱
  - Moderate 👍
  - High carbon ⚠️

### 🔹 Advanced Features
- Outfit footprint calculator 👗
- Eco Champion (best sustainable item) 🏆
- Search functionality 🔍
- Favorite items ❤️
- Dark mode 🌙
